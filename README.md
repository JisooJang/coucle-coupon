# mycoupon-jwt-app
MyCoupon Rest Spring boot App using JWT authentication / authorization. 

## Using
- framework : Spring boot 2.2.6 
- build tool : gradle 6.3
- dependency :
```
Spring boot data JPA
Spring boot starter security
java-jwt
lombok
h2-database
```
- test : Junit4, JUnitParams

## How to Build
```
> gradlew build
```
